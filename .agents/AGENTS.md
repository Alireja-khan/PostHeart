# Project UI/UX Guidelines

## Minimalist & Boundless Design
The user strongly prefers a highly modern, ultra-minimalist, boundless, and interactive UI/UX style. 

When creating or updating UI components for this project, always adhere to the following principles:
- **No Extra Backgrounds or Cards:** Avoid wrapping content in restrictive "cards" with different background colors. UI elements should flow directly on the main global background for perfect color consistency.
- **Boundless Canvas:** Let typography and content dictate the layout rather than rigid boxes or borders. Inputs and text areas should feel organic and expand naturally.
- **High Interactivity & Micro-animations:** Rely heavily on smooth `framer-motion` animations, clever hover states, and dynamic components (e.g. tools that appear contextually or scale smoothly on click).
- **Clean Typography:** Prioritize beautiful typography (using the project's fonts) with high contrast. Let the text itself act as the primary structural element.
- **Floating Contextual Tools:** Tools, menus, and buttons should be minimal, elegantly grouped (like pill-shaped floating docks), and placed out of the way until needed. Avoid native HTML select boxes or bulky default UI elements.
