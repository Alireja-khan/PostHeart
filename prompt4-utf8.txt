### **The Secret Sauce: `transform-origin**`

When you eventually drop your real SVG code in, the most important CSS property you will use is `transform-origin`. If you want an arm to wave, it needs to rotate from the *shoulder*, not from the middle of the arm. `transform-origin` tells the browser where the "joint" is.

Here is your starter code. You can copy and paste this directly into a new HTML file to see it in action.

### **2. The CSS (Layout & Keyframes)**

This CSS handles the three-column layout and the continuous animation loops.

```css
/* --- BASE LAYOUT --- */
body {
    margin: 0;
    background-color: #f8f9fa;
    display: flex;
    justify-content: center;
    align-items: center;
    min-height: 100vh;
}

.animation-container {
    display: flex;
    justify-content: space-between;
    align-items: center;
    width: 90%;
    max-width: 1200px;
}

.character-wrapper {
    width: 250px;
}

.character-svg {
    width: 100%;
    height: auto;
}

/* --- CENTER TRACK STYLING --- */
.center-track {
    flex-grow: 1;
    display: flex;
    flex-direction: column;
    align-items: center;
    position: relative;
}

.track-line {
    width: 80%;
    height: 4px;
    border-bottom: 4px dashed #bdc3c7;
    margin-top: -30px; /* Pulls line under the bird */
}

.bird {
    width: 80px;
    animation: hoverBird 3s ease-in-out infinite;
}

/* --- ANIMATION KEYFRAMES --- */

/* 1. Bird Hovering */
@keyframes hoverBird {
    0%, 100% { transform: translateY(0px); }
    50% { transform: translateY(-15px); }
}

/* 2. Boy's Shirt Waving in the Wind */
.boy-shirt {
    /* Set the "joint" to the top of the shirt */
    transform-origin: 100px 140px; 
    animation: waveShirt 4s ease-in-out infinite;
}
@keyframes waveShirt {
    0%, 100% { transform: skewX(0deg); }
    50% { transform: skewX(-5deg); }
}

/* 3. Boy's Head Nodding */
.boy-head {
    transform-origin: 100px 135px; /* Base of the neck */
    animation: subtleNod 6s ease-in-out infinite;
}
@keyframes subtleNod {
    0%, 100% { transform: rotate(0deg); }
    30% { transform: rotate(3deg); }
    70% { transform: rotate(-2deg); }
}

/* 4. Girl's Leg Shifting (Waiting) */
.girl-leg {
    transform-origin: 90px 280px; /* The Knee/Hip joint */
    animation: shiftLeg 3s ease-in-out infinite;
}
@keyframes shiftLeg {
    0%, 100% { transform: rotate(0deg); }
    50% { transform: rotate(10deg); }
}

/* 5. Girl's Hijab Fluttering */
.hijab-tail {
    animation: flutterHijab 2s ease-in-out infinite;
}
@keyframes flutterHijab {
    0%, 100% { transform: translateX(0) scaleY(1); }
    50% { transform: translateX(5px) scaleY(1.05); }
}

```
